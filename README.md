📈 Stock Price Prediction using Machine Learning

Overview

This project predicts stock closing prices using Machine Learning. The model is trained on historical stock data and learns the relationship between market features to estimate future prices.


Dataset

The dataset contains historical stock market information with the following attributes:

1. Date

2. Open

3. High

4. Low

5. Close

6. Adj Close

7. Volume


Technologies Used:

1. Python

2. Pandas

3. NumPy

4. Matplotlib

5. Seaborn

6. Scikit-learn


Model:

The project uses Linear Regression, a supervised learning algorithm used to predict continuous values.

Input Features: Open, High, Low, Volume

Target Variable: Close price


Workflow:

1. Load dataset

2. Perform Exploratory Data Analysis (EDA)

3. Visualize stock trends and correlations

4. Split dataset into training and testing sets

5. Train Linear Regression model

6. Predict stock prices

7. Evaluate model using MSE and R² score

8. Visualizations

9. Stock price trend graph

10. Correlation heatmap

11. Actual vs Predicted price graph


Conclusion:

The project demonstrates how machine learning can analyze historical stock data and predict future price trends.
